# MatrixEngine ドキュメント

MatrixEngine は、HMI（Human Machine Interface）開発を支援するツールです。  
画面を構成する要素（GUI Object）を中心に設計し、  
デザイナーとエンジニアが役割分担しながら開発を進めることを想定しています。

本ページでは、MatrixEngine に興味を持った方が  
製品の考え方や使い方のイメージをつかめるよう、  
公式ドキュメントを整理して公開しています。
